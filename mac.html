# mac快捷键

chrome
cmd + opt + I 开发者控制台
cmd + opt + U 查看源码
cmd + W 关闭当前标签页

照片路径
个人文件夹—>照片图库－－》，右键，显示包内容打开文件夹，Master目录为照片


cmd + space 输入法切换

cmd + f3 显示桌面
cmd + 上下箭头  上级／下级目录

浏览器
cmd + 上箭头 回到顶部
cmd + 1/2/3 切换不同tab
cmd + l 定位输入地址栏

enter 重命名文件
cmd + c 复制文件
cmd +s 保存
cmd + z 撤销
cmd + opt + v 剪贴文件
cmd + del 删除文件
cmd + 1/2/3 不同方式查看文件夹
cmd + shift + h 打开用户文件夹

窗口快捷键
cmd + w 关闭窗口
cmd + n 新建窗口
cmd + ctrl + f 全／取消全屏
cmd + ` 循环切换相同程序窗口
cmd + tab 切换不同程序窗口

终端快捷键
cmd + T 新建标签
cmd + w 关闭标签
cmd + l 清屏
ctrl + c 中断输入
open path 终端打开目录
进入‘系统偏好设置’->'键盘'->'快捷方式'->‘服务’ 勾选 新建位于文件夹位置的终端标签 ， 新建位于文件夹位置的终端窗口。
